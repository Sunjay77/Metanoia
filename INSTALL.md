# Metanoia - Modern Todo App

Keep track of what matters with **Metanoia**, a beautiful and minimal todo app built with React.

## ✨ Features

- 📱 **Modern Minimal Design** - Clean, distraction-free interface
- ✅ **Easy Task Management** - Add, complete, and clear tasks
- 💾 **Auto-Save** - Tasks saved locally on your device
- 🌙 **Dark Mode Support** - Automatically adapts to your system theme
- ⚡ **Fast & Lightweight** - Responsive performance
- 🎨 **Beautiful UI** - Indigo-themed professional design

## 📥 Installation

### For Android Users

#### Download APK

1. Go to [Releases](https://github.com/Sunjay77/Metanoia/releases)
2. Download the latest `Metanoia.apk` file

#### Install on Your Device

1. **Enable Unknown Sources:**
   - Settings → Security → Unknown Sources → Toggle ON
   - (Location varies by Android version)

2. **Transfer APK to your phone** (via USB, email, or messaging app)

3. **Install the APK:**
   - Open file manager
   - Navigate to Downloads folder
   - Tap `Metanoia.apk`
   - Tap "Install"
   - Wait for installation to complete

4. **Launch the app:**
   - Find "Metanoia" in your apps
   - Tap to open
   - Start organizing!

### For Web Users

Visit: https://metanoia.app (coming soon)

Or build locally:

```bash
npm install
npm run dev
```

## 🚀 Quick Start

1. **Add a task** - Type in the input field and press "Add"
2. **Mark complete** - Check the checkbox next to a task
3. **Delete** - Click the ✕ button
4. **Clear completed** - Click "Clear completed" to remove done tasks

## 🛠️ Tech Stack

- **Frontend:** React 19 + TypeScript
- **Build:** Vite
- **State Management:** Zustand
- **Mobile:** Cordova + Android SDK
- **Styling:** Modern CSS with variables

### Troubleshooting

**"Unknown Sources" blocked the installation?**

- Go to Settings → Security → Unknown Sources
- Toggle the switch ON
- Try installing again

**App won't open after installation?**

- Try clearing the app cache: Settings → Apps → Metanoia → Clear Cache
- Reinstall the app

**Tasks not saving?**

- Your device may have storage restrictions
- Check device storage isn't full
- Try uninstalling and reinstalling

Need more help? Open an issue on GitHub!
